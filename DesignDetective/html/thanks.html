Welcome to Design Detective !

Design Detective is a student project which aim to provide help to web designers (and designers as a whole) by retrieving design cues on a specific website.

Using Design Detectie is easy: Click the big button, the extension will output the font(s) used, color palette(s) and the url to import the font from google font website, (if applicable).

You can also select some text, right click to open the contect menu, and select "Get font". Design Detective will find a font for you.
